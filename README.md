# P3K Care Landing Page

P3K Care adalah platform edukasi interaktif tentang Pertolongan Pertama Pada Kecelakaan (P3K). Website ini dirancang untuk memberikan informasi dan panduan medis dasar dengan antarmuka yang modern, dinamis, dan responsif (Mobile-Friendly).

## Fitur Utama

- **Navigasi Dinamis:** Navbar yang secara otomatis menyorot bagian aktif ketika menggulir, lengkap dengan blur effect modern (Glassmorphism).
- **Mobile Responsive Drawer:** Menu navigasi layar penuh pada perangkat mobile untuk akses yang lebih intuitif.
- **Section Materi (Grid Card):** Kartu pembelajaran dengan efek kemiringan 3D saat disentuh (Hover).
- **Animasi Scroll:** Elemen akan muncul secara berurutan dan halus melalui Intersection Observer API.
- **FAQ Interaktif:** Sistem accordion yang dirancang untuk menjawab pertanyaan yang paling sering diajukan.
- **Pop-up Modal Edukatif:** Detail langkah penanganan pada topik kritis seperti Pingsan, Tersedak, dll tanpa perlu pindah halaman.
- **Form UI yang Responsif:** Desain input *floating label* dengan tampilan modern dan feedback animasi yang informatif.
- **Micro-Animations & Parallax:** Interaktivitas latar belakang yang bergerak mengikuti pergerakan mouse pada desktop.

## Teknologi

Website ini dibuat dengan menggunakan dasar-dasar web standar tanpa kerangka kerja/framework (Vanilla).

- **HTML5** (Semantik)
- **CSS3** (Variabel Native, Flexbox, CSS Grid, Transisi, Keyframes)
- **Vanilla JavaScript** (Akses DOM, IntersectionObserver, Custom Animations)
